## 项目简介

`copyright-crawler` 是一个基于 Go 语言编写的网页自动化脚本。通过使用 [chromedp](https://github.com/chromedp/chromedp) 库，该脚本能够自动化登录中国版权保护中心（ccopyright.com.cn），在通过验证后提取鉴权 Token，并遍历获取完整的省、市、区（县）三级行政地区数据，最终以 JSON 格式输出结果。

## 主要特性

- **自动化网页交互**：自动填充账号密码并模拟点击登录。
- **半自动化人机验证**：默认非无头模式（headless=false），保留人工操作窗口以应对滑块等验证码机制。
- **网络请求拦截**：通过监听 Chrome DevTools Protocol (CDP) 请求，无缝拦截并提取认证 Token（或从 LocalStorage 获取）。
- **异步数据获取**：结合 JS 注入，树状遍历 API 接口获取并拼接省、市、区三级数据。

## 环境要求

- **Go 版本**: >= 1.26
- **本地环境**: 操作系统中需安装 Google Chrome 或 Chromium 浏览器。

## 快速开始

### 1. 配置账号信息

打开项目目录下的 [`main.go`](main.go) 文件，找到如下变量并修改为您在版权中心的账号和密码：

```go
// 1. 设置目标 URL 和凭证
loginURL := "https://register.ccopyright.com.cn/login.html"
username := "YOUR_USERNAME" // 替换为你的用户名/手机号/邮箱
password := "YOUR_PASSWORD" // 替换为你的密码
```

### 2. 下载依赖

在项目根目录下执行以下命令，下载所需的依赖包：

```bash
go mod tidy
```

### 3. 运行项目

执行以下命令运行爬虫程序：

```bash
go run main.go
```

### 4. 运行流程

1. 程序启动后，会自动打开一个可视化的 Chrome 浏览器窗口，并导航至登录页。
2. 程序自动输入账号密码并点击登录。
3. **人工介入**：若页面弹出滑块验证码或图形验证码，**请在浏览器中手动完成验证**。
4. 验证通过后，程序将自动等待页面跳转并接管后续操作。
5. 程序自动跳过引导、模拟点击对应菜单，最终拦截并提取到 Token。
6. 调用底层数据 API 接口进行数据拼装，在终端输出最终的结构化 JSON 数据。
